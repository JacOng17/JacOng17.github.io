# Jacob's Blog

## Run using `bundle exec jekyll serve`

**To start**

`bundle exec jekyll serve` then navigate to [localhost:4000](http://localhost:4000) to view the site.

**To end**

`ctrl+c`

---

## Strawman template

**Blog Posts** _(comment out any unused line with #)_

```
---
layout: post
title:  ""
author: Jacob
tags: [ 1, 2 ]
categories: [ 1, 2 ]
image: assets/images/
description:
featured: (true/false)
hidden: (true/false)
rating: 4.5 (out of 5)
beforetoc: ""
toc: (true/false)
---
```
